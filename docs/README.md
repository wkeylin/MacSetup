## 💡 使用小贴士

### 📖 阅读建议
- 用浏览器的搜索功能 (Ctrl/Cmd + F) 快速找到相关内容
- 每个文档都有详细的目录，可以跳转到感兴趣的部分

- 重要操作前先备份配置
- 遇到问题不要慌，MacSetup 设计得很安全
